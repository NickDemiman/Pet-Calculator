 поле объект, нужное для дальнейшего наследования и удобного хранения в коллекциях. Например, если у нас есть класс Car, который наследуется от класса Vehicle. Тогда, при поиске по Car мы можем обратиться к Vehicle, чтобы получить всю информацию о нем. В случае с объектами, нужно передать адрес объекта, чтобы вызвать метод GetClass (). В C# нет наследования. Это означает, что каждый объект имеет свой собственный уникальный адрес, и нет возможности обратиться к нему по имени. Но существует множество способов сделать это.
